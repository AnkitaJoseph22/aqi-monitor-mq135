# AQI Monitoring System using Arduino Uno and MQ135

## Project Overview
This project is an Air Quality Index (AQI) monitoring system that uses an MQ135 gas sensor connected to an Arduino Uno to collect air quality data. The collected data is processed using Python and machine learning models to predict AQI levels and detect anomalies. The results are displayed on a simple web-based dashboard.

---

## Objectives
- Measure air quality using MQ135 sensor
- Collect real-time data from Arduino
- Process and analyze data using Python
- Predict AQI using trained machine learning models
- Display results on a web interface

---

## Technologies Used
- Arduino Uno
- MQ135 Gas Sensor
- Python
- Machine Learning (Scikit-learn)
- HTML (Frontend)

---

## Project Structure
app.py Main application file
CODE.py Core AQI processing logic
collect_data.py Script to collect data from Arduino
train_model.py Script to train machine learning model
test_serial.py Serial communication testing
index.html Frontend dashboard
*.pkl Trained model and preprocessing files
README.md Project documentation


---

## Working Principle

1. The MQ135 sensor detects air quality and sends analog data to the Arduino Uno.
2. The Arduino transmits the data to the system via serial communication.
3. Python scripts read and preprocess the data.
4. Machine learning models analyze the data to predict AQI and detect anomalies.
5. The results are displayed on a web dashboard.

---

## How to Run the Project

1. Connect the Arduino Uno with the MQ135 sensor.
2. Upload the Arduino code to the board.
3. Run the data collection script:
   python collect_data.py
4. Start the main application:
   python app.py
5. Open the index.html file in a browser to view results.
